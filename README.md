# 🚀 Nail Nuri Portfolio

This is my personal portfolio website built with **Next.js**, **TypeScript**, and **Chakra UI**.

It showcases my projects, skills, and experience as a frontend developer.

## 🌐 Live Demo

👉 https://nurinail.me

## 🛠️ Tech Stack

- Next.js 16
- React 19
- TypeScript
- Chakra UI
- Framer Motion

## ⚙️ Getting Started

Clone the project and run locally:

```bash
git clone https://github.com/nurinail/nailnuri-portfolio.git
cd nailnuri-portfolio
bun install
bun run dev
